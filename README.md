# The Indian Policy Review (IPR)

An independent, youth-led strategic policy think tank and digital research forum mapping Indic perspectives on geopolitics, national security, and governance.

## Technologies Used
* HTML5
* CSS3 (Vanilla Custom Styles)
* JavaScript (ES6+ Vanilla)

## Project Structure
* `index.html` - Semantic structure containing the main homepage, sections, and submission portal.
* `styles.css` - Custom styling rules, colors, typography, card layouts, and animations.
* `app.js` - Search triggers, dropdowns, navigation toggles, and live search filtering.
